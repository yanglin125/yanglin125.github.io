#财务数据

##（省）当前省分城市充值金额接口

  • 地址：Finances/rechargeRank
  • 参数：
    名称|类型| 介绍|案列
    —-|:—-:|—-:
    timetype| int | 1本日，2本周，3本月，4本年|timetype：1
    city_id| int| 410000河南|city_id:410000
    city_type| int| 2|city_type:2

#####请求案列
ajax({url:”/Finances/recharge_rank”,
data:{timetype:3,city_id:410000,city_type:2}}).done((function(data){
if(data.code==1){ } }
}))

#####返回编码
code|message| 处理
—-|:—-:|—-:
1| 成功| 填充echarts数据
500| 未登录访问| js跳转登录页面

#####返回数据格式
